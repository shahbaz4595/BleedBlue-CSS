Introduction

BleedBlue CSS is an open source UI component library that provides pre-styled UI components that can be integrated in your website and can be customized as per your requirements.

BleedBlue CSS uses pre defined classes to style reponsive and accessible components and the theme is inspired by Indian Cricket Team.
